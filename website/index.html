How It Works

Get started in just a few simple steps

1

Add the Integration to Telex

First, add the Server Monitor integration to your Telex workspace using the integration configuration URL.

https://telex-server-monitor.umoru.tech/integration-config
2

Run the Setup Command in Telex

After adding the integration, run the setup command in your Telex channel to initialize the monitoring.

/setup-monitoring
3

Install the Agent on Your Server

Copy the installation command provided by Telex and run it on your server with root/sudo access.

curl -sSL https://telex.example.com/install/<unique-token> | sudo bash
4

Verify the Installation

Check that the monitoring agent is running correctly on your server.

sudo systemctl status telex-monitor
5

Start Receiving Alerts

That's it! You'll now receive real-time alerts and periodic reports in your Telex channel whenever your server needs attention.
